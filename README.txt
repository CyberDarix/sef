# CrustGame - Extreme Gaming Performance Optimizer v1.0.0-alpha

## 📊 Project Overview
CrustGame is a multi-language hybrid project designed to optimize gaming performance through intelligent system resource management.

### **Binaries Included:**
- **CrustUI.exe** - Windows Presentation Foundation (WPF) UI for monitoring and control
- **CrustWatchdog.exe** - Background watchdog service for continuous system optimization

### **System Requirements:**
- Windows 10 / Windows 11 (x64)
- .NET 6.0 Runtime or .NET Desktop Runtime
- Administrator privileges (for optimization features)

### **Quick Start:**
1. Extract the ZIP archive
2. Run CrustUI.exe to launch the graphical interface
3. Click "Gaming Mode: ON" to activate optimization
4. Monitor real-time system metrics in the dashboard

### **Features:**
- Real-time CPU, RAM, GPU usage monitoring
- DPC Latency measurement
- System stability tracking
- One-click Gaming Mode activation
- Reset to defaults option

### **Architecture:**
- **Backend:** Multi-language (C++, C#, Rust, Assembly, Fortran, PowerShell)
- **Frontend:** Windows WPF (C#)
- **Watchdog:** Autonomous monitoring service (C#)
- **Build System:** CMake, MSBuild, PowerShell

### **Development:**
Built with Microsoft Visual Studio 2026, .NET SDK 10.x, and cross-platform build automation.
