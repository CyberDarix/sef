INSTALLATION INSTRUCTIONS:
===========================
1. Download: CrustGame-v1.0.0-alpha.zip
2. Extract to desired location (e.g., C:\Program Files\CrustGame\)
3. Install .NET 6.0 Runtime if not already present
4. Run CrustUI.exe to launch the graphical interface
5. Optional: Run CrustWatchdog.exe as a background service

USAGE GUIDE:
============
CrustUI.exe:
- Launch application
- Monitor real-time system metrics in dashboard
- Click "Gaming Mode: ON" to activate optimizations
- Click "Reset Defaults" to restore original settings
- Status bar shows current system health

CrustWatchdog.exe:
- Run as background service for autonomous monitoring
- Automatically adjusts system parameters based on load
- Logs performance metrics to console output
- Press Ctrl+C to stop service

KNOWN ISSUES / LIMITATIONS:
==========================
• .NET 6.0 is end-of-life (out-of-support) - consider upgrading to .NET 8.0+ in future
• C++ compiler toolset adjustment required (v142 used instead of v143)
• GPU monitoring requires compatible graphics drivers
• DPC Latency reading may be unavailable on some systems
